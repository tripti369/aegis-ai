# aegis-ai
## About the Project
Aegis AI Chatbot is an AI-powered chatbot developed using **FastAPI** and **Groq Llama 3.3**. The chatbot allows users to ask questions and receive intelligent responses through REST APIs. The project focuses on building a clean backend, integrating an LLM, and providing complete testing and documentation.

## Project Overview
Aegis AI Chatbot is a backend application that enables users to interact with a Large Language Model (LLM) through REST APIs. The chatbot is powered by the **Groq API** using the **Llama 3.3 70B Versatile** model.
The project focuses on clean architecture, API testing, documentation, and easy deployment.
